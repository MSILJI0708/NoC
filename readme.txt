# KBO 시뮬레이터

1. KBO 리그의 남은 경기 결과에 따른 전체 순위를 시뮬레이션합니다.


2. NumPy + Numba + Streamlit 기반으로 제작되었고, 웹 브라우저에서 쉽게 사용할 수 있습니다.


3. 사용하는 코드는 gpt plus 의 도움을 받아 만들어졌습니다.


4. 원본 데이터는 kbo 기록실에서 이번 시즌, 현재까지의 기록과 각 팀간의 상대전적을 바탕으로 구성됩니다. 시즌이 바뀌면 최신 시즌 기록을 반영하며, 팀명이 바뀌면 바뀐 팀명을 매치업에 반영하셔야 합니다.


5. 출력은 원본 데이터를 바탕으로 모든 경우의 수를 가정해 각 팀별로 해당 매치업이 끝났을 때 몇 위에 랭크되는지, 또한 각 순위별 비율을 테이블에 표시합니다.




## 실행 방법

1. 의존 패키지 설치:

(사용할 폴더의 bash에서 설치)

pip install -r requirements.txt

requirements.txt 에서 포함하는 모듈들은 아래와 같습니다.
NumPy, Numba, pandas를 활용하여 빠르고 정확하게 결과를 예측하며, Streamlit을 통해 웹 인터페이스로 제공합니다.
파이썬에서 xml 을 처리하기 위해 lxml 모듈도 사용하고 있으니, 이 5개의 모듈이 포함돼있는 requirements.txt 파일을 설치하고 사용하시면 됩니다.

---

## 주요 기능

- 전체 순위 시뮬레이션: 남은 경기의 모든 승/무/패 조합을 계산하여 팀별 순위 확률을 예측
- 상대전적 tie-breaker 반영: 동일 승률일 경우 상대전적에 따라 순위 조정
- Streamlit 기반 웹 UI: 브라우저에서 직접 시뮬레이션 실행
- 빠른 계산 속도: Numba로 연산 최적화, joblib을 통한 병렬 처리

---

## 사용된 기술 스택

- Python 3.11+
- pandas
- numpy
- numba
- joblib
- streamlit
- lxml (웹 데이터 파싱용)

## 사용 방법

1. bash (터미널, 파워쉘 등) 을 열고 아래 코드를 입력합니다.

git clone https://github.com/MSILJI0708/kbo-simulator.git

cd kbo-simulator


2. streamlit run app.py 를 입력하면, 자동으로 사이트가 열립니다.

3. app.py 에서 설정된 매치업대로 시뮬레이션 경기를 진행합니다. 매치업의 사이즈에 따라 작업 시간은 상이할 수 있습니다.
(매치업을 바꾸고 싶다면, app.py에서 직접 매치업을 수정하시면 됩니다.)
